Fitness

A SwiftUI fitness tracking application that allows users to record workouts, monitor activity, and visualize fitness progress through interactive charts and statistics.

Features

* Track running, walking, and cycling workouts
* Real-time workout monitoring
* Route tracking with MapKit
* Workout history and detailed statistics
* Distance, calories, and step count analytics
* Daily goal tracking
* Interactive charts and visual insights
* Local data persistence

Technologies

* SwiftUI
* MVVM
* Core Data
* Core Location
* Core Motion
* MapKit
* Swift Charts
* Combine

Installation

Clone the repository:

git clone https://github.com/PoorneshMV/Fitness.git

Open in Xcode:

cd Fitness
open Fitness.xcodeproj

Run the application on an iPhone Simulator or a physical device.
